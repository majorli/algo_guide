简单的基础算法
==============


本章所述均为简单的基础算法，无需使用递归等高阶技巧，也不涉及分治、动态规划等算法设计技巧，通常用三层以内的嵌套循环即可解决问题。但这些算法往往为复杂算法的基础构件，也是初级算法竞赛考试的常见考点。

本章分为6个部分，每个部分讲解数个典型算法，包括以下内容。

1. 简单数值算法
2. 简单在线统计
3. 简单字符串处理
4. 递归与二分法

   * 递归：基本概念
   * Hanoi塔
   * 递归的错误用法：Fibonacci数列
   * 二分法：基本概念
   * 二分搜索
   * k位数算法
   * 中位数算法

5. 基本排序算法

   * 排序算法的相关概念和基础知识
   * 选择排序
   * 插入排序
   * 二分归并排序
   * 快速排序
   * 桶排序
   * 计数排序
   * 基数排序

6. 简单排列组合

   * 生成排列和组合
   * Pascal三角形（杨辉三角形）：组合数打表
   * 生成随机乱序
