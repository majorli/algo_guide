简单的基础算法
==============


本章所述均为简单的基础算法，无需使用递归等高阶技巧，也不涉及分治、动态规划等算法设计技巧，通常用三层以内的嵌套循环即可解决问题。但这些算法往往为复杂算法的基础构件，也是初级算法竞赛考试的常见考点。

本章分为6个部分，每个部分讲解数个典型算法，包括以下内容。

1. 简单数值算法
2. 简单在线统计
3. 简单字符串处理
4. 递归与二分
5. 基础排序算法
6. 其他基础算法

   * 生成排列和组合
   * Pascal三角形（杨辉三角形）：组合数打表
   * 生成随机乱序
   * 模拟(1)：排座椅（洛谷P1056）
   * 模拟(2)：玩具谜题（洛谷P1563）
   * 模拟(3)：字符串的展开（洛谷P1098）
   * 模拟(4)：生活大爆炸版石头剪刀布（洛谷P1328）
   * 模拟(5)：迎春舞会之数字舞蹈（洛谷P1538）
   * 模拟(6)：花生采摘（洛谷P1086）
   * 模拟(7)：时间复杂度（洛谷P3952）
   * 模拟(番外篇)：tic-tac-toe游戏(I)
