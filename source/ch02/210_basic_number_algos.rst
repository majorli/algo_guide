简单数值算法
------------


数值计算是计算机算法永恒的主题，也是应用数学的一个重要研究领域。从最基础的质数判断到复杂的高精度计算，都是算法竞赛的必备知识。

本节列举的是常用的基本数值算法，大多为基础数论算法，它们是更复杂的数值算法和其他高级算法的基石，包括下面这些内容。

1. 欧几里得辗转相除法：最大公因数、最小公倍数以及互质判断
2. 质数判断、埃氏筛法、欧拉线性筛法打质数表
3. 中国余数定理
4. 整数反转和回文数
5. 有理数的超高精确度小数表示及四舍五入
6. 快速幂
